Setup:

1. You need a webserver in order to perform the tests. Here you can install python and use following command in the Testen-Verifizieren-Validieren/ folder:
python -m http.server &

2. Open a browser and execute the tests. Use this URL:
http://localhost:8000/dojotoolkit/util/doh/runner.html?testModule=excentos.tests.module&registerModulePath=excentos,../../excentos
